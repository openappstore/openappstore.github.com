Open App Store

The Open App Store is an open-source cross-platform app store. It's a way to discover new apps for your computer.

We expect organisations to have their own app stores and prioritise apps that are important to them. Our first goal is to work with the Raspberry Pi community to build an app store for the education sector.

Currently the app store is in development. We encourage developers to get involved and download the current version.

Frequently Asked Questions

Why do we need another app store?

There already many propriety app stores, but the companies running these app stores have complete control over the content. Maybe that doesn't fit in your beliefs or the needs of your organisation.

Mozilla are working on their own app store. We really like what they are doing and there is a lot of overlap with these projects so we would like to work with them and any other organisation passionate about open standards and moving the web forward.

How do I download it?

Currently we are still working on our first version. Join the mailing list to find out more, or download the source code.

What programming languages will be supported?

At first we are focusing on apps built in HTML, JavaScript and CSS. We intend to support JavaScript apps built with Node.js or simple client-side only apps.

Will you support language x, y or z?

Maybe. It's up for discussion.

Will the apps run offline?

Yes, this is a requirement for any app.

Will there be any app review process?

Yes. It will be a completely transparent process; mainly to make sure apps are 1) safe and 2) able to run on the supported operating systems.

What operating systems will be supported?

At first, Microsoft Windows, Apple OSX and Fedora Linux. We plan to be as inclusive as possible. Please get involved to make sure we support your favourite operating system.

Does my app need to be open source to be included in the app store?

This is up for discussion.

Do you have any plans to support other programming languages other than JavaScript?

We are looking into it. Please join in the discussion if you are a developer and willing to help.
